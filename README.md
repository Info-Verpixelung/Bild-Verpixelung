# Bild-Verpixelung

## Frontend starten (bessere Kompatibilität für alle Platformen)

> Wichtig: **Nicht** frontend starten durch öffnen von index.html als Datei im Browser


Folgendes ausführen:
```bash
cd frontend
npm install
npm run dev
```
---
Note: API Requests könnten vorerst **nicht/nicht korrekt** funktionieren
